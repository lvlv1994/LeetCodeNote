# 14. Longest Common Prefix

用第一个string座位prefix，如果之后的不stratsWith这个的话，就substring prefix

```text
class Solution {    public String longestCommonPrefix(String[] strs) {        if (strs.length == 0) {            return "";        }        String pre = strs[0];        for (int i = 1; i < strs.length;i++) {            while (!strs[i].startsWith(pre)) {                pre = pre.substring(0,pre.length()-1);            }        }        return pre;            }}
```
