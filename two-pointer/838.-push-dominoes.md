# 838. Push Dominoes

First pass, from left to right, only count distance of current index to previous 'R'. Save all distance in dist array  
Second pass, from right to left, count distance of current index to previous 'L'. If lDist &lt; rDist \(dist\[i\]\), current cell should be 'L', if lDist == rDist, current cell should be '.'  


```text
class Solution(object):    def pushDominoes(self, dominoes):        """        :type dominoes: str        :rtype: str        """        lst = list(dominoes)        dist = [0]*len(lst)        rDist = None        for i in range(len(lst)):            val = lst[i]            if val == 'R':                rDist = 0            elif val == 'L':                rDist = None            elif rDist != None:                rDist += 1                dist[i] = rDist                lst[i] = 'R'        lDist = None              for i in range(len(lst)-1,-1,-1):            val = dominoes[i]            if val == 'L':                lDist = 0            elif val == 'R':                lDist = None            elif lDist != None:                lDist += 1                if lDist < dist[i] or lst[i] =='.':                    lst[i] = 'L'                elif lDist == dist[i]:                    lst[i] = '.'        return ''.join(lst)                
```
