# 777. Swap Adjacent in LR String

两个指针，因为'L' 只能往左边换，p1 应该大于等于p2,‘R‘只能往右边换，同理，一直找到不是x的为止，然后比较，suppose应该相等，

```text
class Solution(object):    def canTransform(self, start, end):        """        :type start: str        :type end: str        :rtype: bool        """        p1,p2 = 0,0        while p1 < len(start) and p2 < len(end):            while p1 < len(start) and start[p1] == 'X':                p1 += 1            while p2 < len(end) and end[p2] == 'X':                p2 += 1            if p1 == len(start) and p2 == len(end):                return True            if p1 == len(start) or p2 == len(end):                return False            if start[p1] != end[p2]:                return False            if start[p1] == 'L' and p2 > p1:                return False            if start[p1] == 'R' and p2 < p1:                return False            p1 += 1            p2 += 1        return True
```
