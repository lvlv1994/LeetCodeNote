# 389. Find the Difference

python 不能 XOR 但是可以转换成ord 然后 chr一下

```text
class Solution:    def findTheDifference(self, s, t):        """        :type s: str        :type t: str        :rtype: str        """        code = 0        for char in s+t:            code ^= ord(char)        return chr(code)
```
