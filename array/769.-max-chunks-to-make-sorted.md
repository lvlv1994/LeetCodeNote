# 769. Max Chunks To Make Sorted

amazing

```text
class Solution(object):    def maxChunksToSorted(self, arr):        """        :type arr: List[int]        :rtype: int        """        maximum = [0 for i in range(len(arr))]        maximum[0] = arr[0]        for i in range(1,len(arr)):            maximum[i] = max(maximum[i-1],arr[i])        count = 0        for i in range(len(arr)):            if maximum[i] == i:                count += 1        return count
```
