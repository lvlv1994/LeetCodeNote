# 31. Next Permutation

solution说的很清楚

```text
class Solution:    def nextPermutation(self, nums: List[int]) -> None:        """        Do not return anything, modify nums in-place instead.        """        if len(nums) <= 1:            return nums        i = len(nums) - 1                #从后往前找到第一个小于后面一个数的值，就是凹下去的值        while i > 0 and nums[i] <= nums[i - 1]:            i -= 1        # 把后面的list全部交换 因为通过之前的比较 我们已经可以确定之后的list是        #递减数列 那么把它整个swap过来就会得到一个更小的递增数列        self.swapList(nums, i, len(nums) - 1)        #如果这不是一个递减数列的话进行操作 如果是递减数列 直接swaplist就可以了        if i != 0:            j = i            #找到后面的list大于这个凹的点的最小值 换过来就是下一次排列            while nums[j] <= nums[i-1]:                j += 1            self.swap(nums, j, i-1)        return nums        def swap(self, nums, i, j):        tmp = nums[i]        nums[i] = nums[j]        nums[j] = tmp        def swapList(self, nums, i, j):        while i < j:            self.swap(nums, i, j)            i += 1            j -= 1
```
