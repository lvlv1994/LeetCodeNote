# 849. Maximize Distance to Closest Person

好多corner case：

```text
class Solution(object):    def maxDistToClosest(self, seats):        """        :type seats: List[int]        :rtype: int        """        j = -1        result = 0        for i in range(len(seats)):            if seats[i] == 1:                if j == -1:                    result = max(result,i)                else:                    result = max(result,(i-j)/2)                j = i            elif len(seats) == i + 1:                result = max(result,i-j)        return result   
```
