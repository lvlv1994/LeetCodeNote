# 442. Find All Duplicates in an Array

十分有趣的trick，用list本身做hash，如果是负的，说明被访问过。

```text
class Solution(object):    def findDuplicates(self, nums):        """        :type nums: List[int]        :rtype: List[int]        """        result = []        for num in nums:            if nums[abs(num)-1] < 0:                result.append(abs(num))            else:                nums[abs(num)-1] = -nums[abs(num)t-1]        return result
```
