# 144. Binary Tree Preorder Traversal

有三种解法

1. DFS

   值得注意的是 python是Object references are passed by value, 就是说在我们传入result的时候，所有result都指向一个memeory, the function and caller use the same object in memory, but accessed through different variables,  所以我们传入的result不会因为dfs栈的弹出而变化

   ```text
   class Solution:    def preorderTraversal(self, root: TreeNode) -> List[int]:        result = []        self.helper(root, result)        return result    def helper(self, root, result):        if root is None:            return        result.append(root.val)        self.helper(root.left, result)        self.helper(root.right, result)
   ```

   or 更加简便的运用分治算法

   ```text
   return [] if not root else [root.val]+self.preorderTraversal(root.left)+self.preorderTraversal(root.right)
   ```

2. iterative



   ```text
   class Solution:    def preorderTraversal(self, root: TreeNode) -> List[int]:        if root is None:            return         stack, result = [root], []        while stack:            node = stack.pop()            result.append(node.val)            if node.right is not None:                stack.append(node.right)            if node.left is not None:                stack.append(node.left)        return result
   ```
