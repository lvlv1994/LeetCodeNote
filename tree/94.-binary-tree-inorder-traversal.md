# 94. Binary Tree Inorder Traversal

1. Recursion

   ```text
   class Solution:    def inorderTraversal(self, root: TreeNode) -> List[int]:        return [] if root is None else self.inorderTraversal(root.left) + [root.val] +  self.inorderTraversal(root.right)
   ```

2. Iterative

   ```text
   class Solution: def inorderTraversal(self, root: TreeNode) -> List[int]: result,stack = [], [] curr = root    #这个while是一个trick 因为stack为空 curr有可能不为空 循环需要继续    while stack or curr:        while curr:            stack.append(curr)            curr = curr.left        curr = stack.pop()                //这里是可以操作的地方        result.append(curr.val)        curr = curr.right    return result
   ```
