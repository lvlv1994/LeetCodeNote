# 100. Same Tree

1. Recursion:
   1.   ```text
      class Solution:    def isSameTree(self, p: TreeNode, q: TreeNode) -> bool:        if p is None and q is None:            return True        #哈哈注意不能直接else        if p is None or q is None:            return False        return p.val == q.val and self.isSameTree(p.left, q.left) and self.isSameTree(p.right, q.right)
      ```
2.  Iterative
   1. 就是level order traveral 逐个比较啦
