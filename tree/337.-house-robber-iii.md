# 337. House Robber III

是一个dp问题，运用分治 返回下一层抢或者不抢的最大值 

```text
class Solution:    def rob(self, root):        """        :type root: TreeNode        :rtype: int        """        not_rob, rob = self.helper(root)        return max(not_rob, rob)    def helper(self, root):        if not root:            return 0, 0        if not root.left and not root.right:            return 0, root.val        left_not_rob, left_rob = self.helper(root.left)        right_not_rob, right_rob = self.helper(root.right)        //这一层不偷, 下一层随便偷        not_rob = max(left_rob, left_not_rob) + max(right_rob, right_not_rob)        //这一层偷, 下一层只能不偷        rob = left_not_rob + right_not_rob + root.val        return not_rob, rob
```
