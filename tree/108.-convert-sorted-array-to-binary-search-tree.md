# 108. Convert Sorted Array to Binary Search Tree

这道题应该用二分法，二分这个array，然后中序遍历+divide and conqure生成tree

```text
class Solution:    def sortedArrayToBST(self, nums: List[int]) -> TreeNode:        return self.helper(nums, 0, len(nums) - 1)    def helper(self, nums, low, high):        if low > high:            return None        mid = (low + high) // 2        node = TreeNode(nums[mid])        node.left = self.helper(nums, low, mid - 1)        node.right = self.helper(nums, mid + 1, high)                return node
```
