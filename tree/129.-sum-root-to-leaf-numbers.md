# 129. Sum Root to Leaf Numbers

和113差不多 记得不要算current再传进去 这样会更新current 要利用immutable的特性

```text
class Solution:    def sumNumbers(self, root: TreeNode) -> int:        self.result = 0        if not root:            return self.result        self.helper(root, root.val)        return self.result    def helper(self, root, current):        if root.left is None and root.right is None:            self.result += current        if root.left:            //这里不要用current = current* 10 + root.left.val            self.helper(root.left, current* 10 + root.left.val)        if root.right:            self.helper(root.right, current* 10 + root.right.val)
```
