# 102. Binary Tree Level Order Traversal

只要记住 利用qsize这个小trick就可以啦 很简单的一道题哈哈哈

```text
class Solution:    def levelOrder(self, root: TreeNode) -> List[List[int]]:        result = []                if root is None:            return []        import queue        q = queue.Queue()        q.put(root)                while q.qsize() != 0:            size = q.qsize()            tmp = []            for i in range(size):                node = q.get()                tmp.append(node.val)                if node.left is not None:                    q.put(node.left)                if node.right is not None:                    q.put(node.right)            result.append(tmp)                return result
```
