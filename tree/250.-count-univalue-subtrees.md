# 250. Count Univalue Subtrees

因为涉及到左右子树判断的问题，所以理所当然使用divide and conqure 注意每次返回True 或者False定义是否是uni tree就可以啦, 

```text
class Solution:    def countUnivalSubtrees(self, root: TreeNode) -> int:        self.result = 0        self.helper(root)        return self.result    def helper(self, root):        if root is None:            return True        left = self.helper(root.left)        right = self.helper(root.right)                if left and right and (not root.left or root.left.val == root.val) and (not root.right or root.right.val == root.val):            self.result += 1            return True        return False            
```
