# 101. Symmetric Tree

1. Recursive

   1. 注意每次传进去的是左子树的左子树和右子树的右子树就可以啦 vice versa

      

   ```text
   class Solution:    def isSymmetric(self, root: TreeNode) -> bool:        return self.helper(root, root)    def helper(self, left, right):        if left is None and right is None:            return True        if left is None or right is None:            return False        return left.val == right.val and self.helper(left.left, right.right) and self.helper(left.right, right.left)
   ```

2. Iterative

记住加入None进行一起比较 确保正确性

```text
class Solution:    def isSymmetric(self, root: TreeNode) -> bool:        if root is None:            return True        import queue        q = queue.Queue()            q.put(root)                while q.qsize() != 0:            size = q.qsize()            tmp = []            for i in range(size):                node = q.get()                if node is not None:                    tmp.append(node.val)                    q.put(node.left)                    q.put(node.right)                else:                    tmp.append('None')            if tmp != tmp[::-1]:                return False        return True
```
