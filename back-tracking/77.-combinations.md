# 77. Combinations

比subset多加了一步判断

```text
class Solution:    def combine(self, n: int, k: int) -> List[List[int]]:        self.result = []        self.helper(n, k, [], 1)        return self.result    def helper(self, n, k, ans, pos):        if len(ans) == k:            self.result.append(ans[:])            return        for i in range(pos, n + 1):            ans.append(i)            self.helper(n, k, ans, i+1)            ans.pop()
```
