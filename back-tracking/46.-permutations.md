# 46. Permutations

permutation是每次从0开始

```text
class Solution:
    def permute(self, nums):
        """
        :type nums: List[int]
        :rtype: List[List[int]]
        """
        self.visited = []
        self.ans = []
        self.result = []
        self.helper(nums)
        return self.result
    def helper(self,nums):
        if len(nums) == len(self.ans):
            self.result.append(self.ans[:])
            return
        for i in range(len(nums)):
            if i in self.visited:
                continue
            self.visited.append(i)
            self.ans.append(nums[i])
            self.helper(nums)
            self.ans.pop()
            self.visited.pop()
        return
```
