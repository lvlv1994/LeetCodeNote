# 17. Letter Combinations of a Phone Number

这是有一个理解backtracking的机会 for循环里是这个数字对应的字母 每一次recursive都取下一个数字 再找每一个字母

```text
class Solution:    def letterCombinations(self, digits: str) -> List[str]:        KEYBOARD = {    '2': 'abc',    '3': 'def',    '4': 'ghi',    '5': 'jkl',    '6': 'mno',    '7': 'pqrs',    '8': 'tuv',    '9': 'wxyz',}         if not digits:            return []        self.result = []        self.helper(KEYBOARD, [], digits)        return self.result    def helper(self, KEYBOARD, ans, digits):        if len(digits) == len(ans):            self.result.append(''.join(ans))            return        for c in KEYBOARD[digits[len(ans)]]:            ans.append(c)            self.helper(KEYBOARD, ans, digits)            ans.pop()        return
```
