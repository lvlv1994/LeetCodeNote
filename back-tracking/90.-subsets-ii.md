# 90. Subsets II

这是78的变形 加入 \`if i != pos and nums\[i\] == nums\[i-1\]: continue\` 来判断

比如\[1,2,3,4\]的时候，我们在找完\[1,2\]开头的就要pop出2，i移动到2，找 \[1,3\]开头的，那么换做

\[1,2,2,4\]的时候，如果pos 是1且i==1，证明找\[1,2\]开头的，如果pos == 1 且 i== 2，证明\[1,2\]开头的已经找过了，这个时候 \`nums\[i\] == nums\[i-1\]\`, 这两个数字重复，那再找还是 \[1,2\]开头的，所以我们continue

```text
class Solution:    def subsetsWithDup(self, nums: List[int]) -> List[List[int]]:        self.result = []        nums = sorted(nums)        self.helper(nums, [], 0)        return self.result    def helper(self, nums, ans, pos):        self.result.append(ans[:])        for i in range(pos, len(nums)):            if i != pos and nums[i] == nums[i-1]:                continue            ans.append(nums[i])            self.helper(nums, ans, i + 1)            ans.pop()        return
```
