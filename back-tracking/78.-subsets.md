# 78. Subsets

这是一个典型题型哈哈哈，背一下，按照注释思考更容易

```text
class Solution:    def subsets(self, nums: List[int]) -> List[List[int]]:        self.result = []        self.helper([], 0, nums)        return self.result     // 递归三要素    // 1. 递归的定义：在 Nums 中找到所有以 subset 开头的的集合，并放到 results    def helper(self, ans, pos, nums):         // 2. 递归的拆解        // deep copy        // results.add(subset);        self.result.append(list(ans))        for i in range(pos, len(nums)):             // [1] -> [1,2]            ans.append(nums[i])             // 寻找所有以 [1,2] 开头的集合，并扔到 results            self.helper(ans, i+1, nums)            // [1,2] -> [1]  回溯            ans.pop()        // 3. 递归的出口        return
```
