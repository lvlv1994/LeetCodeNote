# 1359. Count All Valid Pickup and Delivery Options

```text
class Solution:
    def countOrders(self, n: int) -> int:
        result = 1
        modulo = 1000000007
        for i in range(1, n + 1):
            result *= i
            result = result % modulo
            result *= (2 * i - 1)
            result = result % modulo
        return result % modulo
            
```
