# 204. Count Primes

从2开始扫，然后但凡没被相乘扫过的就一定为质数

```text
class Solution:    def countPrimes(self, n):        """        :type n: int        :rtype: int        """        if n < 3:            return 0        prime = [False for i in range(n)]             count = 0        for i in range(2,n):                      if not prime[i]:                              count += 1                       for j in range(2,int(n/i)+1):                                     if i*j < n:                        prime[i*j] = True             return count
```
