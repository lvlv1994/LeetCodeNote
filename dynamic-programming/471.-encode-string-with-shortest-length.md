# 471. Encode String with Shortest Length

三重循环，前两重找i和不同length组合的最短长度，第三重是看前面这么长组合一下看谁短一些

```text
class Solution(object):    def encode_substr(self,dp,s,i,j):        tmp = s[i:j+1]        pos = (tmp+tmp).find(tmp,1)        if pos >= len(tmp):            return tmp        return str(len(tmp)/pos)+'[' + dp[i][i+pos-1] +']'    def encode(self, s):        """        :type s: str        :rtype: str        """        dp = [[''for i in range(len(s))] for j in range(len(s))]        for length in range(1,len(s)+1):            for i in range(len(s)+1-length):                j = i + length - 1                dp[i][j] = s[i:i+length]                for k in range(i,j):                    if len(dp[i][k]) + len(dp[k+1][j]) < len(dp[i][j]):                        dp[i][j] = dp[i][k] + dp[k+1][j]                encoded_string = self.encode_substr(dp,s,i,j)                if len(encoded_string) < len(dp[i][j]):                    dp[i][j] = encoded_string        return dp[0][len(s)-1]
```
