# 397. Integer Replacement

```text
class Solution:    def integerReplacement(self, n):        """        :type n: int        :rtype: int        """        c = 0        while n != 1:            if (n&1) == 0:                n = n >> 1            elif (n == 3) or bin(n+1).count('1') > bin(n-1).count('1'):                n -= 1            else:                n += 1            c += 1        return c
```
