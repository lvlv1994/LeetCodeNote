# 246. Strobogrammatic Number

注意建立0的dic哦

```text
class Solution:    def isStrobogrammatic(self, num):        """        :type num: str        :rtype: bool        """        dic = {'1':'1','6':'9','9':'6','8':'8','0':'0'}        start,end = 0,len(num) - 1        while start <= end:                       if num[start] in dic and dic[num[start]] == num[end]:                start += 1                end -= 1            else:                return False        return True        
```
