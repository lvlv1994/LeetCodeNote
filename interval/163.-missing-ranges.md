# 163. Missing Ranges

 因为所有的missing interval都是start+1 到 end -1, 如果这两个相等， 则返回一个数，如果大于，则返回空，不然正常返回。

```text
class Solution(object):    def findMissingRanges(self, nums, lower, upper):        """        :type nums: List[int]        :type lower: int        :type upper: int        :rtype: List[str]        """        result = []        if nums == []:                        return [self.newInterval(lower,upper)]        ans = self.newInterval(lower,nums[0]-1)                if ans != '':            result.append(ans)        for i in range(1,len(nums)):            ans = self.newInterval(nums[i-1]+1,nums[i]-1)            if ans != '':                result.append(ans)        ans = self.newInterval(nums[len(nums)-1]+1,upper)        if ans != '':                result.append(ans)        return result                    def newInterval(self,lower,upper):        ans = ''        if lower > upper:            return ans        elif lower == upper:            ans += str(lower)            return ans        ans += str(lower) + '->' + str(upper)        return ans
```
